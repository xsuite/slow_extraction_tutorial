# Slow Extraction Tutorial (PIMMS)

Jupyter notebooks and helper files for a hands-on tutorial on slow extraction using
Xsuite.

## Notebooks (run in order)

- `notebook_00_basics.ipynb` – load the lattice, basic tracking/visualization
- `notebook_01_phase_space.ipynb` – phase-space characterization
- `notebook_02_tune_extraction.ipynb` – slow-extraction setup
- `notebook_03_spill.ipynb` – control the spill usint transverse excitation
- `notebook_04_multiturn_twiss.ipynb` – multiturn optics functions
